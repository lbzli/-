# 大部分api文档的基本格式
这是大部分文档的基本格式，中括号内的东西代表可选项，可以填也可以不填
类似的还有 | 符号表示或，即可以填这个，也可以填那个，比如
color：inherit | transparent | <color-value> | <color-keyword>
上面用<>包起来的表示某种类型的值，很多编程语言也是这样（模板类）
类似的还有
foo（a，[b=3]）
表示如果你不传第二个参数，则第二个参数b的默认值为3
然后 | 以及 [] 可以组合使用
[]也可以嵌套使用，比如
bar（a[，b[，c]]）
就表示如果填了b，则可以可选的传入c
但如果b都没传，c也不能传
